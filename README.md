🎨 Color Flipper: Run this in any compiler or browser
A simple, interactive web application built with HTML, CSS, and Vanilla JavaScript that instantly changes the page background color with the click of a button
.✨ FeaturesSimple & Hex Modes: Toggle between predefined CSS color names and randomly generated Hex codes.
Instant Visual Feedback: Displays the exact color code currently active on the screen.
Responsive Design: Clean, minimalist UI that looks great on any screen size.🛠️
 Built With HTML5: Structure and semantic layout.
 CSS3: Styling, typography, and layout centering.
 Vanilla JavaScript: DOM manipulation and random color generation logic.
 🚀 UsageOpen the index.html file in any modern web browser.
 Click the "Flip Color" button to generate a random background.
 The page will update, and the active color's code will be displayed in the header.
 📂 File Structure
 index.html - The main structure of the web page. CSS and JS are merged using style tags (<style>, </style>) and script tags (<script>, </script>)
